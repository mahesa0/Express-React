PENJELASAN APLIKASI : 

cara installasi / running :
-buka XAMPP dan jalankan Apache & MySQL
-buka terminal pada folder backend lalu ketikkan 'npm start'
-buka terminal pada folder frontend lalu ketikkan 'npm start'
-buka web localhost berikut 'http://localhost:3000'
-selesai

**FUNGSI TOMBOL GET USERS PADA MENU HOME SAAT BERHASIL LOGIN ADALAH UNTUK MEREFRESH TOKEN ACCESS LOGIN**
**PADA SAAT LOGOUT, COOKIE OTOMATIS AKAN DIHAPUS**
